# Assignment 5

This folder contains all the warmup programs written in Java. Each program performs the following tasks:

program5_1.java – Check whether a number is even or odd

program5_2.java – Find the maximum number out of two numbers

program5_3.java – Check whether the input year is a leap year or not

program5_4.java – Check whether a number is positive or negative

program5_5.java – Find the maximum number out of three numbers

How to Execute the Programs

Use the following commands in the terminal to compile and execute the .java files:

Compile
javac filename.java

Run
java filename


(Note: Do not include .java while running.)

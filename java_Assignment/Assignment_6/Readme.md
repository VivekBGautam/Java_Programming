# Assignment 6

This folder contains all the warmup programs written in Java. Each program performs the following tasks:

program6_1.java – Takes a name as input from the user and displays it on the screen

program6_2.java – Checks whether the given number is greater than 100 or not

program6_3.java – Checks whether the given number is greater than 100 or not (second variation)

program6_4.java – Multiplies three numbers and handles the case where any number is zero

program6_5.java – Calculates percentage using total marks and obtained marks

How to Execute the Programs

Use the following commands in the terminal to compile and execute the .java files:

Compile
javac filename.java

Run
java filename


Note: While running, do not include .java
